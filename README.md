# `windows-11-disable-group-by.reg` and `windows-11-disable-group-by-clean-cache.reg`

Disable Windows 11 auto group by in explorer.

1. Merge `windows-11-disable-group-by.reg`.
2. Merge `windows-11-disable-group-by-clean-cache.reg`.
3. Restart explorer.
